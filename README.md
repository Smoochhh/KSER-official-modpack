# KSER Official Modpack

Welcome to the official **Kerbal Space Exploration & Research (KSER)** modpack.

This repository contains the CKAN export and configuration used throughout the KSER YouTube series. Every mission, launch, cinematic, and episode is recorded using this modpack unless otherwise noted.

---

## About KSER

Kerbal Space Exploration & Research (KSER) is a cinematic Kerbal Space Program series documenting the growth of a space agency from its first launches to future interplanetary exploration.

Every mission has a purpose.

Every spacecraft has a story.

Every success and every failure becomes part of the agency's history.

## Installation

1. Install Kerbal Space Program 1.12.5.3190
2. Install CKAN.
3. Download the KSER CKAN export from this repository.
4. Import the file into CKAN (`File → Import Mod List`).
5. Allow CKAN to install all required mods.
6. Purchase and download Blackrack's Volumetric Clouds V5
7. In the game directory, remove any existing config packs for EVE (BoulderCo, AVP, SVE, Spectra, 64k clouds, KSRSSVE, etc), remove any existing versions of Scatterer and EnvironmentalVisualEnhancements, then merge your GameData folder with the one included in the zip (drag and drop the included GameData in your KSP directory and they will get merged, do not put GameData inside GameData). !! DO NOT REINSTALL SCATTERER OR EVE VIA CKAN !! 
To stop CKAN from overwriting Scatterer and EVE, open CKAN, locate Scatterer and EnvironmentalVisualEnhancements Redux, then right click -> labels -> held, this will stop CKAN from overwriting them.
If you start the game and the screen is pink that means you mixed files from different versions of Scatterer and EVE.
8. Copy any included configuration files if instructed.
